Sales Dashboard

April 7, 2022 by Armando Ayala

For my first blog entry, it was a no-brainer to start with a simple-yet-powerful Sales Dashboard (Report, for the Power BI most knowledgeable audience). For this, I am using the Contoso dataset, with SQL Server as the source (download available here).
The main reason why I choose to start with this visual is because

  • a) It is (to me, at least) the most used type of Data Visualization, and
  • b) It will serve as the data source of many other modelling and visualization tricks.

Filtering Pane

A key element for a neat and versatile Power BI sales dashboard is to use a filtering pane to optimize your space available and still be able to use dropdown slicers with any dimension. Combine with some bookmarks, visual grouping and a couple of Checkmark icons and voila! You have yourself a fancy filtering pane. (This video from Guy in a Cube will teach you how)



Another element I have found to be really useful is to take advantage of the different levels in the matrix, but instead of drilling down (using them as hierarchical categories), using the double arrow to change between levels without filtering. In the example below, you can change from Subcategories, to product Class to Color etc.
This, as a native functionality of the matrix visual, I have found often overlooked.
